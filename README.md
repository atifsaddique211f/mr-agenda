## Description
ModaResa Backend coding exercise.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

The swagger api is available under `api/swagger`.   
e.g: 
```
http://localhost:3000/api/swagger
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Prisma

```bash
# Create migration files
$ npm run init
```
